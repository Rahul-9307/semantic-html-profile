# Semantic HTML Profile Page

This project is a **personal profile web page** built using **semantic HTML** as part of a **Web Development Internship task**.  
The main goal of this project is to understand and implement proper HTML structure using semantic tags.

## Features
- Uses semantic HTML5 tags (`header`, `nav`, `section`, `article`, `footer`)
- Clean, readable, and well-structured HTML code
- Accessible and SEO-friendly layout
- Beginner-friendly project for learning HTML fundamentals

## Technologies Used
- HTML5 (Semantic Elements)
- Basic CSS (if applied)
